# Video Insights Summary — Building Premium Websites with Claude Code

## Video 1: "Building Beautiful Websites with Claude Code Is Too Easy"
**Key Takeaways:**
- Install the front-end design skill globally — single biggest quality multiplier
- Use CLAUDE.md as a system prompt for project context and rules
- Screenshot loop (Puppeteer) creates self-correcting visual feedback
- Provide brand assets (colors, typography, logos) directly in context
- Reference 21st.dev, Awwwards, Dribbble, Godly for inspiration
- Deploy via GitHub + Vercel auto-deploy pipeline from day one
- Iterative refinement beats massive one-shot prompts
- Start simple, then refine section by section

## Video 2: "7 Techniques That Make Claude Code Websites Look $10,000"
**The 7+1 Techniques:**
1. **Tailwind CSS + shadcn/ui** — professional styling foundation
2. **Screenshot cloning** — paste screenshots of sites you admire
3. **Typography** — Google Fonts, 2 fonts (heading + body)
4. **Gradient backgrounds** — radial gradients with multiple colors
5. **3D graphics** — Three.js and Spline (spline.design)
6. **Animations** — GSAP + Framer Motion/motion.dev
7. **Grid systems** — visual grids and layout polish
8. **CLAUDE.md file** — persistent design instructions

## Video 3: "The 7 Levels of Building ELITE Websites with Claude Code"
**The 7 Levels:**
1. **Basic Prompter** — generic, AI-slop output
2. **Visual Director** — screenshots as references (Awwwards, Godly, Pinterest, Dribbble)
3. **Student** — design vocabulary + skills (UIUX Pro Max skill)
4. **Cloner** — deconstruct and clone professional sites (web fetch HTML/CSS/JS)
5. **Original** — combine multiple references into unique designs + custom AI imagery (MidJourney)
6. **Pro/Composer** — 21st.dev components, micro-interactions, video backgrounds, parallax
7. **Architect** — WebGL, shaders, 3D interactive experiences

**Core Insight:** "You are the creative director. Claude Code is the tool."

## Video 4: "Stop Making Ugly Websites with Claude Code"
**The ANF Framework:**
- **A = Assemble** — gather pre-built components from 21st.dev
- **N = Normalize** — unify colors, fonts, spacing into one coherent system
- **F = Fill** — research the actual product and fill with real content

**Common Mistakes:**
- Default purple/indigo everywhere (Tailwind defaults)
- Generic layouts instantly recognizable as AI-generated
- Overused default fonts
- Inconsistent color palette
- Generic placeholder content

## Video 5: "Claude Code + Nano Banana 2 + FireCrawl = Epic $12k Websites"
**Three-Phase Workflow:**
1. **Intelligence Gathering** — FireCrawl to scrape/audit competitor sites, generate research reports
2. **Website Building** — Nano Banana 2 skill for SEO, accessibility, mobile optimization
3. **Visual Enhancement** — Anti-Gravity for 3D animations, Kling 3.0 for AI video

**Key Tools:**
- FireCrawl — website scraping API for competitor intelligence
- Nano Banana 2 — Claude Code skill for premium website building
- 21st.dev — human-designed UI components
- Anti-Gravity — 3D animation software
- Kling 3.0 — AI image/video generation
